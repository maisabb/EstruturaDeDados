# Comparação de Estrutura de Dados - Lista versus Árvore Binária de Busca

Este projeto tem como objetivo comparar o desempenho de busca entre duas estruturas de dados: **lista encadeada** e **árvore binária de busca (ABB)**. A comparação é feita com base na quantidade de comparações necessárias para encontrar elementos previamente armazenados.

## Estrutura do Repositório
- contagem.c -> Código-fonte principal em C, responsável pelo processo de contagem
- resultados.csv -> Arquivo exemplo com os dados gerados pelo código em C
- grafico.R -> Script em R para gerar gráficos com base nos dados do arquivo '.csv'

## Requisitos para execução
- GCC ou outro compilador C
- R (para geração de gráficos)
  
## Como executar
1. Compilar o código C
2. Executar o programa
3. Gerar o gráfico R (em um ambiente R ou RStudio)
